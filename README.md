# SigLab

为新一代交易者打造的*策略工坊*，我们提供最严肃的金融数据和策略测试工具，最傻瓜的拖拽逻辑块或使用自然语言，策略即刻成型。

---

## Design Demo — 为什么要先做这个页面？

`index.html` 是 SigLab 的 **设计标准展示页（Widget Showcase）**。

在动手写任何业务代码之前，我们先建立一套经过验证的视觉标准——配色、字体、间距、组件。有了这些标准，后续无论构建哪个模块，做出来的东西都不会太差。

### 这个 Demo 确立了什么？

**配色体系** — 适合金融场景的双主题方案：
- Dark Mode（深邃海洋 `#0F172A`）：长时间盯盘不伤眼，低蓝光沉浸体验
- Light Mode（纸质感 `#F2F0E9`）：仿 Kindle 暖色调，不是刺眼的纯白
- 涨跌色、警告色、品牌色都经过 WCAG AAA 对比度校验

**多语言字体选择** — 四种语言各有最佳字体：
- 英文 / 西语：IBM Plex Sans
- 中文：IBM Plex Sans SC
- 日文：IBM Plex Sans JP
- 数字 / 代码：IBM Plex Mono（等宽，表格对齐利器）

**字号与排版规范** — 从标题到最小标签的完整层级：
- 大标题 5xl–7xl、Section 标题 1.5rem、正文 sm–base、标签 10px
- 4px 倍数间距系统，组件圆角统一（卡片 2.5rem、按钮全圆、输入框 0.75rem）

**金融场景组件库** — 用真实数据验证每个组件的实际效果：

| 组件 | 说明 |
|------|------|
| K线图 + 成交量 | TradingView Lightweight Charts，支持主题切换 |
| 权益曲线 | 策略 vs Buy&Hold 双线对比 |
| 回撤分析图 | 红色面积图，直观展示风险 |
| 月度收益热力图 | 5 年 × 12 月色块矩阵，绿盈红亏 |
| 交易列表 | 等宽数字、方向色标、盈亏高亮 |
| 行情卡片 | 迷你 Sparkline + 价格闪烁动画 |
| 策略流程图 | CSS+SVG 模拟节点编辑器，虚线流动动画 |
| AI 聊天气泡 | 打字机效果逐字输出 |
| 雷达图 | Canvas 五维策略 DNA 指纹 |
| 按钮 / 表单 / Toast / 骨架屏 / 进度条 | 基础 UI 原件全覆盖 |

### 如何查看

浏览器直接打开 `index.html`，无需任何构建工具。

可交互验证项：
- 点击右上角切换 **暗 / 亮主题**（所有图表联动）
- 点击 **EN / 中 / ES / 日** 切换四种语言
- 行情卡片每 2 秒模拟价格更新 + 闪烁
- AI 聊天区自动播放打字机动画
- Toast 按钮触发 4 种通知类型
- 进度条 Demo 按钮演示动画
