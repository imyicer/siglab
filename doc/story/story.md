# Project Story: SigLab

## 0. 项目故事
我是一名创业者，也是在金融交易行业工作了很多年，现在我想做一个创业项目，大概的描述是，为交易用户提供一个交易策略的 playground。

**项目功能更详细的描述：**
用户可以在咱们这个平台上面学习策略入门，进阶，一直到高级策略。当然用户也可以创建自己的策略。我们会有简单易用的策略生成工具，我的想法是借助拖拽式命令或行为事件，降低创建策略的门槛，尽量不让用户去编程。当然我们也可以通过接入 AI API 的方法，让用户用自然语言去创建策略，调整策略。当用户完成一个策略后，可以一键测试这个策略。我们支持策略的回测，半年、一年、两年、最多五年的回测，生成一目了然的、生动详细的回测报告页面。当然也支持 forward test，比如策略可以跑个两周或者更久，看未来的表现。总的来说，用户在这上面的交易都是基于一个策略的，哪怕是很简单的策略，进行的模拟交易，资金也是模拟的，我们不提供真实资金交易。我们需要有自己的真实报价源，才能提供接近真实的模拟交易。我们还得有过去 5 年的各种市场报价供客户进行回测。至于我们支持哪些产品的交易，我的想法是常见的交易品种，小品种我们就先不做了。

---

## 1. 我们是谁？
SigLab 是一个为新一代交易者打造的**策略实验场**。我们打破了"量化交易必须写代码"的魔咒，将复杂的金融工程抽象为直观、易用的逻辑画布。

---

## 2. 核心定位
**"量化界的 Canva"** —— 我们不提供交易执行，我们提供策略的诞生地。
* **行业坐标**：介于"初级模拟盘"与"高端 Python 量化平台"之间的真空地带。
* **跨界支持**：同时打通 CFD（外汇、黄金、原油）与 Crypto（数字货币）两大流动性最强的市场。

### 2.1 为什么同时做 CFD + Crypto？
市面上绝大多数量化工具只服务单一市场。但现代交易者往往同时关注外汇、黄金与加密货币。SigLab 从第一天起就统一了这两个流动性最强的市场——用**同一套逻辑画布**构建跨市场策略。这意味着：
* 用户不需要在不同平台间切换，学习成本减半。
* 同一套策略逻辑可以快速复制到不同市场进行验证。
* 未来可以探索**跨市场对冲策略**的构建，这是绝大多数竞品无法提供的能力。

### 2.2 竞品对比定位

| 维度 | 传统模拟盘 | **SigLab** | Python 量化平台 |
|---|---|---|---|
| 上手门槛 | 低 | **低** | 高（需编程） |
| 策略构建方式 | 手动下单 | **可视化拖拽 / AI 自然语言** | 写代码 |
| 回测能力 | 无 / 极弱 | **金融级 (Tick 级数据)** | 专业级 |
| 支持市场 | 单一 | **CFD + Crypto 双市场** | 取决于平台 |
| 目标用户 | 纯新手 | **进阶交易者 / 学习者** | 量化工程师 |
| 学习路径 | 无 | **内置策略学院** | 自学为主 |

## 3. 服务人群
* **CFD 进阶交易者**：有主观交易经验，希望通过回测验证自己"感觉"的准确性。
* **Crypto 原住民**：对波动敏感，希望利用简单的逻辑（如网格、趋势跟踪）实现自动化思维。
* **金融学习者**：希望通过低门槛工具快速上手策略构建的初学者。

---

## 4. 为什么选择我们？
* **零代码生存**：拖拽逻辑块或直接对 AI 说话，策略即刻成型。
* **金融级回测**：底层基于 LEAN 引擎，数据颗粒度达到 Tick 级，拒绝"虚假盈利"报告。
* **纯粹性**：无资金实盘压力，这里是纯粹的逻辑验证与技能提升空间。
* **跨市场统一**：CFD 与 Crypto 共享同一套策略构建工具，学一次用两处。
* **AI 加持**：自然语言描述即可生成策略，AI 还能自动诊断和优化已有策略。

---

## 5. 我们的战略边界
**我们只做策略的诞生地，不做资金的屠宰场。**

SigLab 刻意不接入实盘交易，这不是能力缺失，而是战略选择：
* **对用户**：消除了"急于赚钱"的心理压力，让用户专注于逻辑本身的优化。
* **对市场**：我们与券商/交易所保持**非竞争关系**——我们是他们的上游。一个在 SigLab 上训练好策略的用户，最终会成为券商的优质客户。
* **对监管**：不涉及实盘资金流转，大幅降低合规成本和法律风险。

## 6. 商业模式方向

### 6.1 Freemium（免费增值）
* **免费层**：基础策略构建、有限品种/时间范围的回测、基础模板
* **Pro 层**：完整 5 年回测、全部品种、AI 助手高级功能、Forward Test、策略导出
* **Team 层**：团队协作、策略共享空间、优先技术支持

### 6.2 B2B 白标合作
券商/交易所可以将 SigLab 以白标形式嵌入其平台，作为"策略教育工具"提供给客户，提升客户粘性和留存率。

### 6.3 策略市场（远期）
用户可在平台上发布/购买策略模板，平台按交易抽佣。优质策略创建者可获得收入分成。
