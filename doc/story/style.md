# UI/UX Style Guide

## 1. 核心设计理念
* **Professional yet Vibrant**: 专业感来自于图表的严谨，活泼感来自于 UI 的细节响应和配色。
* **Focus**: 屏蔽复杂的菜单，让用户的视野聚焦在“逻辑画布”上。

## 2. 配色方案 (Color Palette)
* **Primary (深空色系)**: `#0F172A` (背景色，沉稳、专注)。
* **Secondary (科技蓝)**: `#3B82F6` (主要按钮、激活状态)。
* **Accent (生机绿/警示橙)**: 
    * 盈利/买入: `#10B981` (翡翠绿)
    * 亏损/卖出: `#EF4444` (活力红)
    * 辅助色: `#8B5CF6` (紫色，用于 AI 功能，体现智慧感)。

## 3. 字体建议
* **UI 文本**: `Inter` 或 `Plus Jakarta Sans` (现代、易读、年轻化)。
* **数据/代码**: `JetBrains Mono` (体现量化与数字的专业感)。

## 4. UI 元素
* **Card**: 玻璃拟态 (Glassmorphism)，带有微弱的边框描边，增强层次感。
* **Animations**: 使用 Framer Motion，让拖拽块在吸附时有弹簧回馈感。
