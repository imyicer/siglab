# Product Features List

## 1. 策略生成模块 (Core)
* **拖拽编程 (Visual Flow)**:
    * 事件触发器 (OnData, OnOrderEvent)。
    * 指标库 (MACD, RSI, EMA, etc.)。
    * 仓位控制逻辑。
* **AI 助手**:
    * 自然语言描述转策略逻辑。
    * 现有策略逻辑的参数优化建议。

## 2. 回测与分析 (Analysis)
* **回测配置**: 时间范围、初始资金、杠杆倍数、手续费模型。
* **报告中心**: 
    * Equity Curve (权益曲线图)。
    * Drawdown Analysis (回撤分布)。
    * Trade List (详细每笔交易流水)。

## 3. 实时观测站 (Forward Test)
* **模拟实盘**: 部署回测好的策略，连接实时 WebSocker 行情。
* **实时告警**: 当策略触发买卖信号时，推送浏览器通知。

## 4. 学习中心 (Academy)
* **互动教程**: 手把手教用户如何构建第一个“均线交叉”策略。
* **模板库**: 官方提供各资产类别的标准策略参考。
